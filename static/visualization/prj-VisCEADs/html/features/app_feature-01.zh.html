3.1.2. 📜主要、 次要、任选 指标选择器及分页#

用户在此可以从16（原数据共 30）指标（含进口出口等）中，选择主要、 次要、任选 ，分别在三次分页进行探究比较。

举例来说，用户可对产业的 产值 及 环境影响 进行探究比较。

表格中亦列出 百分位排名、百分比、排名、值 等基本统计及信息。

3.1.3. ❓要降多少？主要指标碳排強度#

主要指标如　碳排強度 及 二氧化碳排放 要降多少？

PwC 及 澳恪森研究均指出，世界碳排强度需年均减少17-19%，才能达《巴黎协定》1.5℃目标。

中国2024年《2024—2025年节能降碳行动方案》要求碳排强度（经济） 年减3.9%。

此表交互可以选定指标间的比较。

二氧化碳排放 指标要降多少？

  • 中国2024年已下指标要年减3.9%左右（单位国内生产总值二氧化碳排放，可视为经济年脱碳率 或 经济碳排强度年減率）。见《2024—2025年节能降碳行动方案》。

  • PwC指出，世界碳排强度需年均减少17.2%，才能达《巴黎协定》1.5℃目标。见《2023淨零排放經濟指數報告》（Net Zero Economy Index 2023）。

  • 澳恪森研究发现，世界碳排强度需 年均减少19% ，才能达1.5℃目标。见VisNetZero成果）。包括中国的新兴七国，2030年前需加4倍努力，达到年减9.8-10.3% 的脱碳率，才能达1.5℃目标。近年中国脱碳率表现最好的年份及数据分别为 -7.773%（2015年）、 -7.334%（2016年）、 -6.822%（1980年）、 -6.495%（2014年）、等。最近2021年亦达-4.180%。中国2024年方案指标相比之下并不严苛。
