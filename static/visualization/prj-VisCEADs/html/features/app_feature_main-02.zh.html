2.2. 🖱网络图可供控件#

由投入产出关系的 网络图 构成的主图，可供用户探究以下 产业结构 的投入产出关系，以及多项指标信息。

2.2.1. 示意展示图#

下图仅为展示，完整交互功能见VisCEADs仪表盘。

2.2.2. ℹ️主图的交互功能#

搭配侧边栏交互控制套件（主图左侧），可以调控投入产出网络主图，会随选响应变化。

控制套件包括以下的选择器。

2.2.2.1. A. 年份#

年份 可选 2012 、 2015 、 2017 、及 2017新 。其中 2017新 开始采用较为不同的行业分类。

2.2.2.2. B. 地区#

地区 可选CEADs数据集中有提供之各省市。

2.2.2.3. C. 主要指标#

主要指标可选：

  • CO2排放

  • 总产出

  • 碳排强度

CEADs 主要指标

本网络图可选的主要指标依据为《2024—2025年节能降碳行动方案》：单位国内生产总值二氧化碳排放 指标（碳排强度－每经济单位） 及 单位国内生产总值能源消耗（能源强度－每经济单位） ：

  • CO2排放

  • 总产出

  • 碳排强度 = CO2排放 / 总产出

选取主要指标后，投入产出网络的节点大小会随选响应变化。适合用来比较各 节点（行业） 的排放和产出如何发展与污染有效脱钩（包括关键指标 碳排强度 ），打断经济发展与环境污染水平之间的关系。

网络图节点大小随选响应特性

本网络图节点大小会随选响应变化，是用来比较节点，比较省份，比较年份，比较指标的利器，可搭配选择器运用。

2.2.2.4. D. 网络参数调整#

调整可视化参数方面，可以调整下列两者参数：

  1. 引力：网路各节点往中心吸

  2. 斥力：网路各节点弱或无关系的互斥

引力-斥力 模型为重力模型，假定两节点间的关系愈强，则吸引力愈强，若弱关系甚至无关系，或有互相排斥力。

2.2.2.5. E. 侧边栏 额外信息栏#

侧边栏 另外提供以下额外信息栏：

  • 交互功能说明

  • 感谢说明

  • 澳恪森其它项目及信息

2.2.3. ❓要降多少？主要指标碳排強度#

主要指标如　碳排強度 及 二氧化碳排放 要降多少？

PwC 及 澳恪森研究均指出，世界碳排强度需年均减少17-19%，才能达《巴黎协定》1.5℃目标。

中国2024年《2024—2025年节能降碳行动方案》要求碳排强度（经济） 年减3.9%。

此表交互可以选定指标间的比较。

二氧化碳排放 指标要降多少？

  • 中国2024年已下指标要年减3.9%左右（单位国内生产总值二氧化碳排放，可视为经济年脱碳率 或 经济碳排强度年減率）。见《2024—2025年节能降碳行动方案》。

  • PwC指出，世界碳排强度需年均减少17.2%，才能达《巴黎协定》1.5℃目标。见《2023淨零排放經濟指數報告》（Net Zero Economy Index 2023）。

  • 澳恪森研究发现，世界碳排强度需 年均减少19% ，才能达1.5℃目标。见VisNetZero成果）。包括中国的新兴七国，2030年前需加4倍努力，达到年减9.8-10.3% 的脱碳率，才能达1.5℃目标。近年中国脱碳率表现最好的年份及数据分别为 -7.773%（2015年）、 -7.334%（2016年）、 -6.822%（1980年）、 -6.495%（2014年）、等。最近2021年亦达-4.180%。中国2024年方案指标相比之下并不严苛。
